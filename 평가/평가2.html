HTML
html언어는 홈페이지의 컨텐츠를 입력할 때 사용하는 언어입니다. html언어는 정보의 맥락에 맞게 알맞은 태그를 사용하는 것이 종요합니다.
CSS
CSS언어는 홈페이지의 스타일을 적용하는 언어입니다. CSS언어를 통해서 HTML언어로 만들어진 태그를 선택하고 알맞은 스타일을 적용함으로써 홈페이지의 디자인을 완성할 수 있습니다.
JAVASCRIPT
Javascript언어는 홈페이지의 동적인 컨텐츠를 입력할 수 있으며, 사용자의 이벤트에 반응하여 사용자와 상호작용하는 기능을 넣을 때 주로 사용하는 언어입니다. javascript를 통해 우리는 홈페이지와 상호작용을 할 수 있습니다.
